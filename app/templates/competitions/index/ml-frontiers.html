Objectives

The ML Frontiers competition is designed for machine learning enthusiasts to showcase their skills in building, testing, and deploying cutting-edge models. Compete against others in solving complex challenges and demonstrate your expertise in predictive analytics.

Participants will be required to build machine learning models for specific use cases. Instead of submitting a CSV file, users will submit the actual model itself in a supported format such as .pkl. A sample Colab notebook with instructions on building and exporting the model will be provided.

Rules

  • Participants must register before the competition starts.
  • Each participant will be given a machine learning problem to solve within a specified time frame.
  • Participants must use the provided Colab notebook to build and export their models.
  • Models must be submitted in a supported format such as .pkl.
  • Instructions about how the model should work, the type of input it expects, and the type of output it gives will be provided in the competition description.

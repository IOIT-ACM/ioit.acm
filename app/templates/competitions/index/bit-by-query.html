Objectives

Bit by Query is a competition where students can test their Database Management System (DBMS) skills. It provides a platform for participants to write SQL queries and perform operations on a dashboard to solve a variety of DBMS-related questions. This competition is designed to evaluate and challenge your practical abilities in SQL and database management.

Rules

  • Participants will write and execute SQL queries directly on the competition platform.
  • Each query is run against a specific database designed for the problem.
  • Your submission is judged automatically by comparing the output of your query against the correct, expected result.
  • A solution is considered correct only if it is syntactically valid and the output (including column names and data order) exactly matches the expected result.
  • For problems involving INSERT, UPDATE, or DELETE queries:
    • Provide two queries separated by a semicolon.
    • The first query should manipulate data, and the second should display the result.
  • For problems involving SELECT queries, provide a single query as your solution.
  • Use MySQL syntax for all queries.
  • Problem status indicators:
    • Blue box: Solved problems.
    • Gray box: Unsolved problems.
    • Green box: Active problems.
  • When the countdown ends, a Start button will appear. Once the competition time is up, you will be redirected to the "Times Up" screen.
  • If you encounter any issues, refresh the page.
  • Enjoy the competition and give it your best!
